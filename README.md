# Adidas Sales Analysis

## Overview
This project analyzes Adidas sales performance in the US market using Python.  
The analysis focuses on data cleaning, exploratory data analysis (EDA), visualization, and business insights to understand sales trends, product performance, regional differences, and channel effectiveness.

## Objectives
- Analyze overall sales performance
- Identify top-performing product categories
- Compare sales across regions and sales channels
- Generate business insights and recommendations

## Dataset
The project uses the Adidas US Sales dataset for analysis.  
The dataset includes transaction information such as:
- invoice date
- retailer
- region, state, city
- product category
- price per unit
- units sold
- total sales
- operating profit
- sales method

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Workflow

### 1. Data Cleaning
- Removed duplicate records
- Handled missing values
- Standardized text columns
- Converted date and numeric columns to proper data types
- Detected and removed outliers using IQR
- Prepared clean data for analysis

### 2. Exploratory Data Analysis
- Monthly and quarterly sales trends
- Revenue distribution by region
- Product category performance
- Online vs offline sales comparison
- Retailer revenue and profit comparison
- Seasonal sales analysis

### 3. Visualization
Built multiple charts to present insights clearly, including:
- line charts
- bar charts
- heatmaps
- bubble charts
- radar charts
- waterfall charts

## Key Insights
- Sales dropped significantly in 2020 and recovered strongly in 2021
- The West region contributed the highest revenue
- Women’s Apparel and Men’s Street Footwear were among the strongest product categories
- Online sales played an important role, but offline channels still showed strong performance
- Retailer performance varied significantly across partners
